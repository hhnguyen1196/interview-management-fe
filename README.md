# Interview Management Fe

Dự án này được tạo ra bằng [Angular CLI](https://github.com/angular/angular-cli) version 18.2.21.

## Cài đặt


Trước khi chạy dự án, hãy đảm bảo rằng bạn đã cài đặt **Node.js version v.20.12** và **Angular** trên máy.  
Tại thư mục gốc của dự án, chạy lệnh sau để cài đặt toàn bộ dependencies:
```bash
   npm install
```
## Máy chủ phát triển

Chạy `ng serve` để khởi động máy chủ phát triển. Truy cập vào `http://localhost:4200/`.  
Ứng dụng sẽ tự động tải lại khi bạn thay đổi bất kỳ tập tin nguồn nào.
